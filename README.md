Latihan membuat aplikasi catatan yang dapat menulis, mengubah, dan menghapus catatan. Aplikasi ini menggunakan javascript node js, hapi, &amp; eslint.

- ## Cara Penggunaan

- Buka http://notesapp-v1.dicodingacademy.com/
